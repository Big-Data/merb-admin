merb-admin by sferik

MerbAdmin is a Merb plugin that provides an easy-to-use interface for managing your data.

Dependencies

merb-slices

Install

In your app, add the following dependency to `config/dependencies.rb`: dependency "sferik-merb-admin", "0.3.6", :require_as => "merb-admin" Add the following route to `config/router.rb`: slice(:MerbAdmin, :name_prefix => nil, :path_prefix => "", :default_routes => false) Then, run the following rake task: rake slices:merb-admin:install
